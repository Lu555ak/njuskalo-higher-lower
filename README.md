# Njuškalo | Higher/Lower
![alt text](https://github.com/Lu555ak/njuskalo-higher-lower/blob/main/banner.png)

A fast-paced guessing game where you compare two cars from Njuškalo and guess which one costs more!

## How to Play

1. Two cars appear with their specs
2. Click on the car you think is more expensive
3. Build your streak with correct guesses
4. One wrong guess ends the game!

## Features

- Real car listings from Njuškalo
- Beautiful gradient UI with smooth animations
- Mobile-friendly responsive design
- High score tracking with local storage

## Built With

- HTML5, CSS3, Vanilla JavaScript
- No frameworks or dependencies
- Works in any modern browser

## Quick Start

```bash
# Clone and open
git clone https://github.com/yourusername/higher-lower-cars.git
cd higher-lower-cars
# Open index.html in your browser - that's it!
```
